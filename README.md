# DoubbleGDP Test

This App enables users to create a todo list, and updates the list when it done.

## Project Setup

* Clone this repo
* Run `bundle install`
* Run `rails db:migrate`
* Run `npm install`
* Run `rails server`
* Run `./bin/webpack-dev-server`


## Ruby version
* 2.7.1

## Database
* sqlite

## Testing
* run `rspec`
* run `npm run test`
